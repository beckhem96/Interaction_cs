# CS Visual Lab

CS Visual Lab is an interactive Korean-first web app for learning computer science concepts through step-by-step visual execution.

The product direction includes data structures, algorithms, SQL query behavior, network communication, and operating-system behavior. The current implementation uses React, TypeScript, Vite, React Router, Vitest, and SVG-first visualizations, but future features may use other languages, renderers, simulators, or external libraries when they improve the learning flow and are justified in the plan.
